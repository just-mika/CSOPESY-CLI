========================================================================
CSOPESY-CLI - Command Line Interface Project
========================================================================

By: Amon, Mikaela

This simple project simulates the `nvidia-smi` command in the CLI.

------------------------------------------------------------------------
Prerequisites
------------------------------------------------------------------------
- Visual Studio 2026 (for VS-based execution)
- GCC compiler with C++20 support (for terminal-based execution)

------------------------------------------------------------------------
How to Run
------------------------------------------------------------------------

1. Via Visual Studio 2026
   -----------------------
   - Open 'CSOPESY-CLI.slnx' in Visual Studio 2026.
   - Set the configuration to either Debug or Release mode.
   - Run or debug the project (F5 or Ctrl+F5).

2. Via Terminal
   ------------
   - Make sure g++ is installed and configured in your system PATH.
   - Open your terminal or command prompt.
   - Navigate to the project directory:
     cd (Project Directory)/CSOPESY
   - Compile the source files using the C++20 standard:
     g++ -std=c++20 *.c -o program
   - Execute the compiled program:
     program.exe

------------------------------------------------------------------------
Notes
------------------------------------------------------------------------
* Ensure all necessary source files (.c / .cpp) are present in the target 
  directory before compiling via the terminal.
